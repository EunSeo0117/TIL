# Tree

## 개념
노드와 간선으로 구성된 비선형 계층적 자료구조이다.  
하나의 루트 노드에서 시작하고, 노드들이 부모-자식 관계를 가지며, 노드들간의 사이클이 없다.

## 특징
- 계층 구조 표현에 적합하다
- 탐색 시간이 보통 O(N)이다
- 재귀적 구조이다
- 연결 그래프이면서 비순환 구조이다.
- N개의 노드를 가지면 간선은 N-1개
- 

## 트리종류
- 일반트리(자식 개수 제한없음)
- 이진트리(자식이 최대 2개)
- 이진탐색트리(BST)
  - 왼쪽 < 부모 < 오른쪽
  - 평균 탐색 O(log N)
  - 최악 O(N)
- 균형 이진 트리
  - AVL Tree
  - Red-Black Tree
- 힙
  - 완전이진 트리
  - 부모 >= 자식(max heap)
  - 부모 <= 자식(min heap)

## 트리 순회 방법
- DFS
  - 전위 순회
  - 중위 순회
  - 후위 순회
- BFS
  - 레벨단위 탐색
  - 큐 사용

## 활용예시
- 파일 시스템
- 조직도
- DOM구조
- 의사결정 트리
- 데이터베이스 인덱스(B-Tree)
- 힙 정렬

## 면접대비 질문

### 트리와 이진트리, 이진탐색트리에 대해 설명해 주세요.
- 트리는 노드와 간선으로 구성된 계층적 비선형 자료구조 이다. 하나의 루트노드에서 시작하며, 부모-자식관계를 가지고, 사이클 없는 연결 그래프 이다.  
- 이진트리는 각 노드가 최대 두개의 자식을 가지는 트리이다. 
- 이진탐색트리(binaryy search tree)는 정렬 규칙을 가지는 이진트리이다. 왼쪽 서브트리의 모든 값은 부모보다 작고, 오른쪽 서브트리의 모든값은 부모보다 크다. 탐색을 빠르게 하기위해 정렬규칙을 추가한 이진트리이다. 


### 이진탐색트리에서 중위 탐색을 하게 되면, 그 결과는 어떤 의미를 가지나요?
이진탐색 트리에서 중위순회를 하면 오름차순 정렬된 값이 된다. 그이유는 왼쪽은 항상 작은값, 오른쪽은 항상 큰 값으로 유지되기 때문이다. 

### 이진탐색트리의 주요 연산에 대한 시간복잡도를 설명하고, 왜 그런 시간복잡도가 도출되는지 설명해 주세요.
| 연산 | 평균       | 최악   |
| -- | -------- | ---- |
| 탐색 | O(log N) | O(N) |
| 삽입 | O(log N) | O(N) |
| 삭제 | O(log N) | O(N) |

탐색, 삽입, 삭제는 루트에서 리프노드까지 내려가는 과정이므로 트리의 높이에 비례하여 최대 O(h)가 되어, 한쪽으로 트리가 편향된 경우에는 높이h만큼의 시간복잡도를 가진다. 양쪽이 균형잡힌 구조일경우 그 높이에 비례하여 log N이 된다.

### 이진탐색트리의 한계점에 대해 설명해주세요.
이진탐색트리는 데이터삽입 순서에따라 구조가 달라지며, 자동균형 유지 기능이 없다. 그래서 트리가 편향되면 탐색 성능이 선형시간으로 하락한다.  
이를 해결하기위한 AVL Tree, Red-Black Tree같은 균형 이진탐색 트리가 존재한다. 

### 이진탐색트리의 값 삽입, 삭제 방법에 대해 설명하고, 어떤식으로 값을 삽입하면 편향이 발생할까요?
삽입은 루트부터 시작하여 작은값은 왼쪽, 큰값은 오른쪽으로 이동시킨다. 삭제는 3가지 경우가 있는데, 리프노드의 경우 그냥 제거하고, 자식이 하나인경우 자식으로 대체, 자식이 두개인경우 오른쪽 서브트리의 최소값, 또는 왼쪽서브트리의 최대값으로 대체한다.  
편향은 정렬된 순서로 삽입할 경우에 발생한다.  1,2,3,4,5를 순서대로 삽입하는 경우 항상 오른쪽으로만 삽입이 되기때문에 한쪽으로 기울어진 트리가 생성된다. 

### 이진탐색트리와 동일한 로직을 사용하면, 삼진탐색트리도 정의할 수 있을까요? 안 된다면, 그 이유에 대해 설명해 주세요.
삼진트리도 정의는 가능하다. 하지만 일반적인 bst와 동일한 개념으로 확장하면 문제가 생긴다. 그이유는 이진트리는 1개의 비교기준으로 2구간을 분리하지만, 삼진트리를 만들게되면 하나의 기준값으로 구간분리가 불가능하게되어 2개의 기준값이 필요하게되기때문에 이진탐색트리의 논리를 그대로 확장하기 어렵다.  
대안으로는 B-Tree, 2-3Tree, 2-3-4Tree같은 다진탐색 트리로 확장해야한다.
